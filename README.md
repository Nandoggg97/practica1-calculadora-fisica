## Descripcion
Este proyecto corresponde a la Práctica 2 de la materia de Estructura de Datos. Se refactorizó la calculadora física desarrollada en la práctica anterior para utilizar los modificadores `ref` y `out`, eliminando el uso de valores de retorno (`return`) y aplicando el paso por referencia en los métodos de cálculo.

## Tecnologias utilizadas

- C#
- .Net
- Visual Studio Code
- Git
- GitHub

## Compilar el proyecto

```bash
dotnet build
```

## Ejecutar el proyecto

```bash
dotnet run
```

## Reflexión

Durante esta práctica aprendí la diferencia entre el paso por valor y el paso por referencia mediante los modificadores `ref` y `out`. También reforcé el uso de Git para trabajar con ramas, realizar commits y fusionar cambios de forma ordenada. Considero que estos conocimientos serán muy útiles como futuro ingeniero en sistemas, ya que permiten desarrollar aplicaciones mejor estructuradas y mantener un control adecuado de las versiones del código.
